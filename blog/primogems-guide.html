🔥 Best Primogem Sources for New Players

1. Archon Quests — Best Primogems/Hour

Archon Quests give 60 Primogems per chapter. Sumeru and Fontaine acts are especially long and rewarding. If you're new and need wishes fast, rush the main story. Each region's Archon chain gives roughly 600-1,000 Primogems total.

2. World Exploration (Chests + Oculi + Sigils)

Each region has hidden chests worth 2-10 Primogems each. You'll find 200-400 chests per region. Inazuma's Electrogrus, Sumeru's Dendroculi, and Fontaine's Hydroculi all reward at least 200 Primogems when you level up the Statues of The Seven.

Best regions for chest farming:

  • Liyue — 1,000+ chests, easy to find
  • Dragonspine — Crimson Agate tree: 500+ Primogems
  • Chasm Underground — Dense chests, ~600 total
  • Sumeru — Massive map, dendroculi rewards stack high

3. Achievements

Hidden achievements often reward 5-20 Primogems. Some category completions give 100. Check the Wonders of the World tab — many are easy to trigger by accident.

💡 Pro tip: Complete "The Catch" fishing quest line in Inazuma. It's free, takes about 2 hours, and the weapon is one of the best 4-star polearms in the game. Side reward: plenty of Primogems along the way.

4. Temporary Events

Every patch has 3-4 events. The flagship event (usually the first one) alone gives 1,000+ Primogems + a free 4-star weapon. Don't skip these — they're time-limited but usually only take 30-60 minutes to clear.

❌ Primogem Traps to Avoid

  • ✖️ Standard Banner — Never spend Primogems here. Use only free Acquaint Fates.
  • ✖️ Weapon Banner — Only if you're a whale. Epitomized Path is designed to drain you.
  • ✖️ Resin Refills — 50 Primogems for 60 Resin is not worth it for F2P.
  • ✖️ Character Banner at low pity — If you're below 70 pity, don't pull just because "one more 10-pull".
  • ✖️ BP + Welkin — Best value if you spend anything, but this guide is for F2P only.

📅 Weekly Checklist

  • ✅ Daily commissions × 7 days
  • ✅ Spend all Original Resin (preferably on boss materials or talent books)
  • ✅ Check event tab for new content
  • ✅ Redeem Parametric Transformer (weekly gadget)
  • ✅ Check the Adventurer's Handbook milestones
  • ✅ One Abyss run (you only need floor 9-10 for easy 300 primogems)

🎯 Should You Save or Spend?

Here's a simple rule: save for characters you actually want, not because a banner is running. Genshin has no FOMO if you understand that every character will rerun eventually.

If you're saving for a specific character:

  • Zhongli / Nahida / Kazuha / Furina — account-defining supports, worth 180 pulls
  • Limited 5-star DPS — pull only if you need their element/role
  • C1/C2 constellations — not worth it for F2P unless it's a game-changer (like Hu Tao C1)
