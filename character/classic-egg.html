📝 Overview

Classic Egg is exactly what it sounds like — the egg you start with, the egg you'll always come back to. Perfectly balanced across all stats, Classic Egg has no weaknesses and no special gimmicks. This makes it an excellent choice for players who want a consistent experience on any map.

🎯 How to Unlock

Available from the start. Classic Egg is the default character for all new players.

💡 Pro Tips

  • Because Classic Egg has no special abilities, focus on mastering the fundamentals — dash timing, jump arcs, and map shortcuts.
  • Use Classic Egg to learn new maps before switching to specialized characters.
  • Classic Egg is a solid pick for Race mode where balanced stats matter more than gimmicks.
