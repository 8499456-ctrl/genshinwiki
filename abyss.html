🏆 T0 Abyss Teams

🔥 雷国队 — Best Budget T0

Raiden + Xingqiu + Xiangling + Bennett

Raiden on-field battery, Xingqiu hydro app, Xiangling off-field Pyro, Bennett ATK buff + heal. All 4-star weapons viable. Core is the energy recharge loop.
Difficulty:⭐⭐
Budget swaps: None (all core, irreplaceable)

🌊 龙芙万钟 — Current Meta King

Neuvillette + Furina + Kazuha + Zhongli

Neuvillette's charged attack hits like a truck, Furina provides team-wide DMG bonus, Kazuha shreds, Zhongli provides shield and stagger resistance.
Difficulty:⭐⭐⭐⭐
Budget swaps: Zhongli → Baizhu/Kirara

💨 神鹤万心 — Freeze Ceiling

Ayaka + Shenhe + Kazuha + Kokomi

Ayaka's burst guarantees crit on frozen enemies, Shenhe's Icy Quills boost damage, Kazuha groups and shreds, Kokomi provides Hydro app and healing. Blizzard Strayer gives 40% free crit rate.
Difficulty:⭐⭐⭐
Budget swaps: Shenhe → Rosaria/Ganyu

🌿 草行久 — Cheapest T0 Team

Nahida + Xingqiu + Kuki

Cheapest T0 team, all C0 viable. Dendro aura + Hydro core + Electro trigger = Hyperbloom, homing missiles. 4th slot is flex.
Difficulty:⭐
4th slot options: Zhongli/Alhaitham/Fischl/Yaoyao

📋 Abyss Team-Building Principles
  • Build teams separately for each half — Counter the specific enemy lineup
  • At least one healer/shielder per team — Survival first; dead characters deal no DPS
  • Ensure energy recharge — High damage is useless if bursts aren't up
  • Elemental shield break — Abyss heralds need the right element to break shields
  • Focus on one team first > spreading resources — Build one T0 team before starting the second
  • 12-3 > 12-1 — Chamber 3 second half is usually the hardest. Prioritize that chamber's needs
